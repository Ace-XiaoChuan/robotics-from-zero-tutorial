# Robotics Learning Notes

本仓库用于记录笔者从零开始学习机器人相关知识的过程与阶段性成果，内容涵盖 C++ 基础复习、现代 C++ 语法、ROS 2 实践以及机器人运动学等方向。

## 项目简介

笔者此前已经学习过经典 C++，但对于指针、取地址、解引用、常引用，以及现代 C++ 中的一些常用语法仍存在理解不够深入或记忆不够清晰的问题。因此，本仓库将相关学习内容进行系统整理，方便后续复习、查阅与持续完善。

同时，本仓库也会逐步记录 ROS 2 相关实践内容，以及机器人学基础知识的学习过程。

## 学习内容

目前已整理的课程内容如下：

| Lesson | 内容 |
|---|---|
| Lesson 1 | 复习引用 `&` |
| Lesson 2 | 复习 Lambda 表达式 |
| Lesson 3 | 复习构造函数、析构函数、独占智能指针 |
| Lesson 4 | 实践 ROS 2 接口 |
| Lesson 5 | 学习 ROS 2 组件 Component |
| Lesson 6-7 | 机器人正运动学与 DH 参数表实践 |

## 学习目标

通过本仓库的持续记录，希望能够完成以下目标：

- 巩固 C++ 基础语法与核心概念
- 理解现代 C++ 中常用但容易混淆的语法特性
- 掌握 ROS 2 的基本开发流程与常用机制
- 建立机器人学基础知识体系
- 将理论学习与实际代码实践相结合

## 适用对象

本仓库主要适合：

- 正在入门机器人开发的学习者
- 希望复习 C++ 与现代 C++ 语法的学习者
- 正在学习 ROS 2 基础内容的学习者
- 希望通过实践理解机器人正运动学与 DH 参数表的学习者

## 说明

本仓库内容以学习记录为主，部分代码和笔记可能会随着理解的深入不断修改和完善。如有错误或不严谨之处，欢迎指出与交流。
